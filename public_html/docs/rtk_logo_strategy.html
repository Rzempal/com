ResztaToKod.pl

System Identyfikacji Wizualnej v1.0

01. Primary Logo (Long)

Wymiar bazowy: 440x60 px
Proporcje: ~7.3:1
Technologia: SVG + CSS Animation

Symbolika & Zastosowanie

Pełna sekwencja narracyjna marki. Przedstawia transformację z nieuporządkowanej struktury (sieć neuronowa/chaos) w liniowy proces.

02. Compact Logo (Short)

Wymiar bazowy: 110x40 px
Proporcje: ~2.75:1
Technologia: SVG (Static/Light)

Symbolika & Zastosowanie

Minimalistyczna wersja do nagłówków mobilnych, sticky menu i znaków wodnych.

03. Favicon (Dark Mode / Transparent)

Tło: Brak (Przezroczyste)
Kolor: Biały (#ffffff)

Opis

Wersja bez tła z białymi konturami. Przeznaczona do wyświetlania na ciemnych tłach.

04. Favicon (Light Mode / Transparent)

Tło: Brak (Przezroczyste)
Kolor: Czarny (#000000)

Opis

Wersja bez tła z czarnymi konturami. Przeznaczona do wyświetlania na jasnych tłach.

05. Favicon (Adaptive / Auto)

Tło: Brak (Przezroczyste)
Kolor: Auto (CSS Media Query)

Zastosowanie Pro

Ten plik SVG zawiera wbudowany styl CSS, który wykrywa ustawienia systemowe użytkownika. Będzie czarny w trybie jasnym i biały w trybie ciemnym.

06. App Icon (Light / White Background)

Tło: Białe (#ffffff)
Linie: Czarne (#000000)
Kształt: Squircle (Nowoczesny)

Zastosowanie

Ikona aplikacji lub profilowa (Social Media) na białym tle. Kształt Squircle (zaokrąglony kwadrat) pasuje do standardów iOS i Android.

07. App Icon (Dark / Black Background)

Tło: Czarne (#0a0a0a)
Linie: Białe (#ffffff)
Kształt: Squircle (Nowoczesny)

Zastosowanie

Ikona aplikacji lub profilowa na ciemnym tle. Idealna do trybu ciemnego lub jako elegancki element marki.
